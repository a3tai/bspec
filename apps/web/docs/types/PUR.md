# PUR: Organizational Purpose

**Domain**: Strategic Foundation

The Purpose document articulates the organization's social impact and stakeholder value beyond profit. It defines the broader positive change the organization creates in the world.

## Metadata Schema

```yaml
---
id: PUR-{identifier}
type: PUR

priority: critical|high|medium|low

depends_on: []

---
```

## Quality Levels

| Level | Requirements |
|-------|--------------|
| Bronze | Clear purpose statement articulated, Primary beneficiaries identified, Basic impact description provided |
| Silver | Comprehensive stakeholder value analysis, Business model alignment demonstrated, Impact measurement framework defined |
| Gold | Full documentation with validation evidence |

## Relationships

**Depends on**: MSN, VAL
