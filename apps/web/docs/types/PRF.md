# PRF: Performance Marketing

**Domain**: Brand & Marketing

The Performance Marketing document defines data-driven marketing strategies focused on measurable outcomes and return on investment. It establishes performance frameworks that optimize marketing spend, maximize conversions, and deliver accountable results through systematic testing, measurement, and optimization across all marketing channels.

## Metadata Schema

```yaml
---

depends_on: []

---
```

## Quality Levels

| Level | Requirements |
|-------|--------------|
| Bronze | Basic performance marketing strategy with key channels and measurement, Simple optimization approach and basic ROI tracking, Basic attribution model and performance reporting |
| Silver | Comprehensive performance marketing strategy with advanced measurement, Structured optimization framework with systematic testing, Active performance management with advanced analytics and attribution |
| Gold | Full documentation with validation evidence |

## Relationships

**Enables**: LED, CON, ROI
