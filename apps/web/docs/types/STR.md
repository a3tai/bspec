# STR: Business Strategy

**Domain**: Strategic Foundation

The Strategy document defines how the organization will achieve its vision and compete in its chosen markets. It articulates the key choices about where to play, how to win, and what capabilities to build.

## Metadata Schema

```yaml
---

depends_on: []

---
```

## Quality Levels

| Level | Requirements |
|-------|--------------|
| Bronze | Clear where-to-play choices defined, Basic how-to-win approach articulated, Core capabilities identified |
| Silver | Comprehensive strategic framework, Detailed competitive analysis integrated, Strategic initiatives mapped and prioritized |
| Gold | Full documentation with validation evidence |
