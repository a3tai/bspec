# MOT: Competitive Moats

**Domain**: Strategic Foundation

The Moats document identifies and analyzes the competitive advantages that protect the organization's market position. Moats are sustainable advantages that make it difficult for competitors to replicate success.

## Quality Levels

| Level | Requirements |
|-------|--------------|
| Bronze | At least 2-3 competitive advantages identified, Basic description of each moat, Customer value explanation for each advantage |
| Silver | Comprehensive moat analysis framework, Strength assessment for each moat, Competitive barrier analysis |
| Gold | Full documentation with validation evidence |
